# しぼログPRO（仮称）

カロリー・PFC・体調・トレーニングまで管理できるダイエットサポートアプリ。
既存の「しぼログ」とは別プロダクト（既存アプリ・既存ユーザーには影響なし）。
販売方針: 買い切りの有料アプリ（サブスク・アプリ内課金なし）。外部APIも不使用で、完全にオフラインで動作します。

## 構成
- React 18 + Vite / PWA / Capacitor 8（Android土台込み）
- データ保存: IndexedDB（日別レコード分割・写真は専用ストアに分離）
- 軽量データはlocalStorageにもミラー保存（二重化）
- バックアップ: JSONエクスポート / インポート（**旧しぼログv3形式にも対応**）/ 復元・リセット前の安全バックアップ / Android実機では日次自動バックアップ（Documents/shibolog_next_backups、7世代）

## ページ構成（7タブ）
📊ダッシュボード（今日の体重・週平均・週間カレンダー・体重/カロリー推移・週ごとの平均）/
🍽️食事（インライン入力・マイミール・本日のメモ・写真は端末保存のみ）/
💪トレーニング（部位チップ・5セット×推定1RM(O'Conner式)・前回記録・当日最高1RM）/
🔥メニュー提案（training-menu-app移植: 月4〜28回・POF・パワー/パンプ・脚分割ルール）/
📋記録（週選択・朝夜体重・絵文字・備考・今週の振り返り）/
📅カレンダー（🌅体重・💪・絵文字表示、日別編集）/
⚙️設定（目標設定・基本情報・旧しぼログ準拠の基本計算・目標体脂肪率計算・データ管理）

## 開発
```
npm install
npm run dev      # 開発サーバー
npm test         # データ層テスト
npm run build    # 本番ビルド
npm run android:sync  # Androidへ同期
```
GitHubへpushすると Actions で debug APK が自動ビルドされます。
